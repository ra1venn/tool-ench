# The Minecraft enchantments optimizer    

This thing will help you get the best enchantments while spending as little experience as possible.  
- **The details on what this thing does:** This is a webpage with some UI that allows you choose the enchantments you want to be on your tool or armor, and then builds an optimized enchantment order, so you can save your levels and experience.

# Getting started
You can go and try it right now with the [site](https://mc.ra1venr.fun/) that i made recently  

or download it for yourself:  

Firstly, you need to download the project and using  
```
npm install
```  
inside the directory. This will download necessary dependencies.  
Next step is running locally with 
```
npm run dev
```
or creating one simple HTML file with 
```
npm run build
```
Using build will create /dist directory with ```index.html``` file in it.
### That's it for now. Hope this one saves someone from boring grinding for levels.
